## Empty Cells

The following example table contains some blank cells in the **Special Food** and **Notes** column.  By default, empty cells are just skipped when rendering the card-like item.  In other words, the list item that corresponds to the empty column will not be there.

If the default behavior is not desired, custom content for empty cells can be provided.  In this demo, `@lucide/svelte`'s `<CircleSlash2 />` (<CircleSlash2 size="1em" />) icon is injected into empty cells and then an opacity of 40% is applied to the cell via the stylesheet imported from the layout component.
